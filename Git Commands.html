Git Commands

git Command

What does it do

git init

Adds a new git repository

git add filename

Adds a new file to the staging area

git status

Lists all staged and unstaged files

git commit -m "description of commit"

Makes a commit (saves changes) with a comment

git checkout HEAD filename

Undoes changes in the working directory

git reset Head filename

Unstages files from the staging area back to the working directory

git reset + first 7 numbers of commit

Resets to a previous commit of choice;

git add .

Adds all files from the working directory to the staging area
