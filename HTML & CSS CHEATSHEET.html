RGB Colors

>

The RGB color system constructs all the colors from the combination of the Red, Green and Blue colors.
The red, green and blue use 8 bits each, which have integer values from 0 to 255.
This makes 256*256*256=16777216 possible colors.
RGB â‰¡ Red, Green, Blue
Each pixel in the LED monitor displays colors this way, by combination of red, green and blue LEDs.
When the red pixel is set to 0, the LED is turned off.
When the red pixel is set to 255, the LED is turned fully on.
Any value between them sets the LED to partial light emission.

HSL Colors

In HTML, a color can be specified using hue, saturation, lightness (and alpha) in the form:

HSLA (hue, saturation, lightness, alpha)

The alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not transparent at all).

Example

hsl(0, 100%, 0%)
hsl(0, 100%, 25%)
hsl(0, 100%, 50%)
hsl(0, 100%, 75%)
hsl(0, 100%, 90%)
hsla(9, 100%, 64%, 0.6)
